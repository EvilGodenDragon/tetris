# 俄罗斯方块项目介绍及架构信息

## 1. 项目概述
**项目背景和价值**:
- 基于 HTML5 Canvas 和原生 JavaScript 实现的经典俄罗斯方块网页游戏。
- 无需安装，通过浏览器即可直接运行，支持跨平台（桌面端、平板和移动端）。
- 提供现代化的界面设计和主题切换功能，增强用户体验。

**核心用户问题**:
- 解决了传统游戏需要下载客户端的问题，实现即开即玩。
- 解决了不同设备适配问题，响应式设计兼容多种屏幕尺寸。
- 提供了本地数据持久化，保存玩家的游戏记录和偏好设置。

**系统功能**:
- **核心玩法**: 完整的俄罗斯方块机制（移动、旋转、加速下落、硬降）。
- **主题系统**: 支持明亮模式和夜间模式切换，自动保存偏好。
- **辅助功能**: 幽灵方块预览落点位置。
- **计分与排行**: 本地存储历史最高分和消除记录，等级随消除行数提升。
- **操作控制**: 
  - 键盘按键控制（方向键、空格、P/R/Enter 等）
  - 触摸按钮控制（移动端虚拟按键）
- **响应式设计**: 完美适配桌面、平板、手机三种设备类型。

**技术亮点**:
- **模块化设计**: 清晰的职责分离，每个模块专注于单一功能。
- **CSS 变量主题**: 优雅的主题切换实现，无需 JavaScript 操作 DOM 样式。
- **Canvas 优化**: 高效的渲染机制，支持主题动态切换。
- **墙踢算法**: 智能的方块旋转偏移修正。
- **响应式设计**: 完美适配桌面和移动设备。

**技术约束和限制**:
- **性能需求**: 依赖浏览器 JS 引擎性能，Canvas 渲染效率较高。
- **安全需求**: 标准 Web 安全规范，无敏感数据上传。
- **已知问题/风险**:
    - 音效和背景音乐尚未实现。
    - 多人对战模式尚未实现。
    - 截图路径在 README 中为占位符，需实际添加。
    - 部分老旧浏览器可能不支持 Touch Events API（但主流现代浏览器均支持）。

## 4. 项目目录结构
**核心模块介绍**:
```
俄罗斯方块/
├── index.html          # 主页面入口，包含 Canvas 容器和触摸控制按钮
├── css/
│   └── style.css      # 样式文件，定义主题变量、响应式布局和触摸控制样式
├── js/
│   ├── constants.js   # 游戏常量配置 (颜色、速度、分数表)
│   ├── piece.js       # 方块类定义 (形状、旋转、颜色)
│   ├── board.js       # 游戏面板逻辑 (网格、碰撞、消除)
│   ├── game.js        # 游戏主控制器 (循环、状态管理、Canvas动态缩放)
│   ├── storage.js     # 本地存储管理 (排行榜、设置)
│   └── main.js        # 入口文件和事件处理 (初始化、输入监听、触摸控制)
├── deploy-to-github.ps1 # GitHub 部署脚本
├── README.md          # 项目说明文档
├── GITHUB_GUIDE.md    # GitHub 相关指南
├── PROJECT_SUMMARY.md # 项目摘要
└── QUICK_START.md     # 快速开始指南
```
- **配置中心**: `js/constants.js` 集中管理游戏平衡性参数 (速度、颜色、分数)。
- **逻辑核心**: `js/game.js` 和 `js/board.js` 构成游戏核心循环。
- **持久化**: `js/storage.js` 独立管理用户数据，解耦业务逻辑。
- **响应式核心**: `css/style.css` 中的媒体查询和 `js/game.js` 中的 resizeCanvas 方法共同实现自适应。

## 5. 移动端适配详情

### 5.1 响应式断点设计

| 设备类型 | 屏幕宽度 | 布局特点 | Canvas方块尺寸 |
|---------|---------|---------|--------------|
| 桌面端 | >768px | 三栏布局，完整操作说明 | 30px（默认） |
| 平板端 | 481-768px | 双栏布局，隐藏左侧说明 | 最大25px |
| 手机端 | ≤480px | 单栏紧凑布局，显示触摸控制 | 最大20px |
| 超小屏 | ≤360px | 进一步缩小字体和间距 | 最小12px |

### 5.2 触摸控制功能

**虚拟按钮布局**（固定在屏幕底部）：
```
[←] [↻] [↓] [→] [⤓]
左  旋  下  右  硬
```

**交互特性**：
- 单击触发：左移、右移、旋转、硬降
- 长按连续：下落按钮支持长按连续触发（100ms间隔）
- 防误触：所有触摸事件阻止默认行为，防止页面滚动
- 自动显示：检测到移动设备或屏幕宽度≤768px时自动显示

### 5.3 动态Canvas缩放算法

```javascript
// 根据视口宽度计算最佳方块尺寸
if (viewportWidth <= 480) {
  // 手机端：预留边距，最大20px
  blockSize = Math.min(Math.floor((viewportWidth - 20) / COLS), 20);
} else if (viewportWidth <= 768) {
  // 平板端：预留边距，最大25px
  blockSize = Math.min(Math.floor((viewportWidth - 40) / COLS), 25);
} else {
  // 桌面端：固定30px
  blockSize = BLOCK_SIZE;
}

// 确保最小可玩尺寸
blockSize = Math.max(blockSize, 12);
```
